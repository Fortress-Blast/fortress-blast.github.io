Fortress Blast adds collectable powerups from the Marble Blast series to compatible maps. Each powerup has a special ability that can be activated for a short amount of time.

You can see your ready powerup on the right side of your HUD in-game. Press the 'Special attack' key to use it. This bind is set to MOUSE3 by default, but you can change this in TF2's Options.

Click the tabs above to learn more about this server mod.

You can host Fortress Blast on your own server by visiting https://github.com/Fortress-Blast. Follow the instructions in the readme of the pinned repository before installing.
